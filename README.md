# Task 3: Dashboard Design

## Objective
Design an interactive dashboard for business stakeholders using Power BI.

## Dataset
Financial Sample dataset (Sales, Profit, COGS, Discounts by Country/Segment/Product, 2013-2014)

## KPIs Used
- Total Sales
- Total Profit
- Profit Margin %
- Total Units Sold

## Features
- 4 summary cards for key totals
- Time-series line chart (Total Sales by Date)
- Bar chart: Total Sales by Product
- Bar chart: Total Profit by Country
- Interactive slicers: Country, Segment, Product
- Consistent orange color theme

## Key Insights
- Total Sales: 118.73M | Total Profit: 16.89M | Profit Margin: 14.23%
- Paseo is the top-selling product by sales volume
- France generated the highest profit among all countries
- Sales showed noticeable fluctuation month-to-month across 2013-2014, with a strong peak toward the end of the period

## Tools Used
Power BI Desktop

## Files
- `Task3_Financial_Dashboard.pbix` — Power BI dashboard file
- `Financial_Sample.xlsx` — dataset used
- `dashboard_screenshot.png` — dashboard preview
- `Task3_Summary.pptx` — PPT summary

## Note
This is a single-page dashboard; a navigation menu was not required for this scope.
